一、增长量X

（一）已知现期B和基期A

增长量（X）=现期（B）−基期（A） \text{增长量（X）}=\text{现期（B）}-\text{基期（A）} 增长量（X）=现期（B）−基期（A）

（二）已知基期A和增长率R

增长量（X）=基期（A）×R \text{增长量（X）}=\text{基期（A）}\times\text{R} 增长量（X）=基期（A）×R

（三）已知现期B和增长率R

  • 基础公式：

增长量（X）=现期（B）1+R×R \text{增长量（X）}=\frac{\text{现期（B）}}{\text{1+R}}\times R 增长量（X）=1+R现期（B）​×R

二、增长率R

（一）一般增长率

  • 基础公式：

增长率（R）=增长量（X）基期（A） \text{增长率（R）}=\frac{\text{增长量（X）}}{\text{基期（A）}} 增长率（R）=基期（A）增长量（X）​

  • 题目：2020年，LED产业总产值为8490亿元；2019年LED产业总产值为7548亿元；
  • 问题：求总产值同比增长率?

    解析

    R=XA=8490−75487548=9427548=10%+2%+0.6%=12.6%\text{R}=\frac{\text{X}}{\text{A}} = \frac{8490-7548}{7548} = \frac{942}{7548} = 10\%+2\%+0.6\%=12.6\%R=AX​=75488490−7548​=7548942​=10%+2%+0.6%=12.6%

（二）间隔增长率

  • 基础公式：

R=R1+R2+R1×R2 R = R_1 + R_2 + R_1 \times R_2 R=R1​+R2​+R1​×R2​

  • 题目：2015年，生产时钟(含钟心)完成产值162亿元，同比下降4.7%，降幅扩大1.3个百分点。
  • 问题：2015年我国钟表全行业生产时钟(含钟心)的产值比2013年增长多少?

    解析

    2015年R1=−4.7%R_1=-4.7\%R1​=−4.7%，降幅扩大1.3个百分点，则2014年R2=−3.4%R_2=-3.4\%R2​=−3.4%
    间隔增长率R=−4.7%−3.4%+(−4.7%)×(−3.4%)=−8%R=-4.7\%-3.4\%+(-4.7\%)\times(-3.4\%)=-8\%R=−4.7%−3.4%+(−4.7%)×(−3.4%)=−8%

（三）比值增长率（平均数的增长率）

  • 题型识别 均/平均 增长率%
  • 基础公式：

R1−R21+R2 \frac{R_1 - R_2}{1+ R_2} 1+R2​R1​−R2​​

  • 题目：2018年前三季度，S省社会物流总额35357.26亿元，同比增长6.4%，2018年前三季度，S省社会物流总费用2682.1亿元，同比增长6.3%;
  • 问题：2018年前三季度，平均每万元社会物流总额产生的物流费用比上年同期?

    解析

    2018年前三季度S省社会物流总费用增长率R1=6.3%R_1=6.3\%R1​=6.3%，S省社会物流总额增长率R2=6.4%R_2=6.4\%R2​=6.4%
    带入比值增长率公式R1−R21+R2=−0.1%1+6.4%>−1%\frac{R_1 - R_2}{1+ R_2}=\frac{-0.1\%}{1+6.4\%}>-1\%1+R2​R1​−R2​​=1+6.4%−0.1%​>−1%，下降了不到1%

乘积增长率-常规

  • 题目：2017 年全国棉花播种面积为 3229.6 千公顷，比 2016 年减少 146.6 千公顷。分地区看，我 国最大的产棉区新疆棉花播种面积比 2016 年增加 157.9 千公顷，增长 8.7%：新疆棉花播种面积占全国的 比例，从 2016 年的 53.5%进步扩大到 2017 年的 60.8%，比上年增长 7.3 个百分点。其他棉区中，2017 年 黄河流域棉花播种面积减少 215.1 千公顷，下降了 24.3%，长江流域棉花播种面积减少 97.0 千公顷，下降 14.9%。
    2017 年全国棉花每公顷单位面积产量为 1698.6 公斤。其中，新疆棉区每公顷单位面积棉花产量增加 88.4 公斤，同比增长 4.4%：黄河流域棉区每公顷单位面积产量增加 22.9 公斤，增长 2.1%：长江流域棉区 每公顷单位面积产量减少 39.5 公斤，下降 3.6%。

  • 问题1：2017 年，新疆棉花总产量同比增速在以下哪个范围内？
    A．不到一成
    B．一成多
    C．两成多
    D．超过三成

    解析

    题型分类：ABRX 类之乘积增长率；
    解析：定位文字材料，已知：产量=面积×单位面积产量，新疆棉花播种面积比 2016 年增长 8.7%，新疆棉区每公顷单位面积棉花产量同比增长 4.4%，利用公式： R=Ra+Rb+Ra×RbR = R_a + R_b + R_a \times R_bR=Ra​+Rb​+Ra​×Rb​可得，2017 年新疆棉花总产量同比增速=（8.7%+4.4%+8.7%×4.4%）＞13%
    答案为 B

  • 问题2：2017 年，长江流域棉区棉花总产量同比约下降：
    A．4%
    B．7%
    C．15%
    D．18%

    解析

    题型分类：ABRX 类之乘积增长率；
    解析：定位文字材料，已知：产量=面积×单位面积产量，长江流域棉花播种面积下降 14.9%，长江流域棉区每公顷单位面积产量下降 3.6%。利用公式： R=Ra+Rb+Ra×RbR = R_a + R_b + R_a \times R_bR=Ra​+Rb​+Ra​×Rb​可得，2017 年长江流域棉区棉花总产量同比增速=-14.9%+（-3.6%）+（-14.9%）×（-3.6%）≈-18%
    答案为 D

乘积增长率-整体和部分

  • 题目：2019年，我国海洋生产总值89415亿元，比上年增长6.2%；2019年，我国海洋第三产业增加值占占海洋生产总值的比重由59.1%上升到60%。
  • 问题：2019年我国海洋第三产业增加值年增长率为?
    A．6.4%
    B．7.1%
    C．7.8%
    D．8.5%

    解析

    题型分类：ABRX 类之乘积增长率；
    解析：已知，海洋第三产业增加值=海洋生产总值×第三产业增加值占海洋生产总值比重，2019 年海洋生产总值增长率=6.2%， 2019 年第三产业增加值占海洋生产总值比重增长率=60−59.159.1≈1.5%\frac{60-59.1}{59.1}\approx 1.5\%59.160−59.1​≈1.5% ，利用公式：R=Ra+Rb+Ra×RbR = R_a + R_b + R_a \times R_bR=Ra​+Rb​+Ra​×Rb​可得，人身险原保险保费收入增速=6.2%+1.5%+6.2%×1.5%=7.8%6.2\%+1.5\%+6.2\%\times1.5\%=7.8\%6.2%+1.5%+6.2%×1.5%=7.8%，答案为 C。

（三）隔年基期

  1. 识别：隔一年，求基期。
  2. 方法：
    • 第一步，求间隔增长率。
    • 第二步，间隔基期=现期1+间隔 r\text{间隔基期}=\frac{\text{现期}}{\text{1+间隔 r}}间隔基期=1+间隔 r现期​
  • 题目：2021 年志哥私房钱有 100 元，同比增速为 10%，2020 年同期增速为5%。
  • 问题：2019 年志哥私房钱为_____元？

解析

给 2021 年求 2019 年，第一步求间隔 r，间隔 r=10%+5%+10%×5%=15.5%r=10\%+5\%+10\%\times5\%=15.5\%r=10%+5%+10%×5%=15.5%，间隔基期=100/（1+15.5%）

四、现期B

  • 基础公式：

现期（B）=基期（A）×(1+R) \text{现期（B）} = \text{基期（A）} \times ( 1 + R ) 现期（B）=基期（A）×(1+R)

现期（B）=基期（A）+增长量（X） \text{现期（B）} = \text{基期（A）} + \text{增长量（X）} 现期（B）=基期（A）+增长量（X）
